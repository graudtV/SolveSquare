◆ getNewData()

double* getNewData ( const char  prompt[],
size_t  nElements

Считывает из стандартного ввода указанное число чисел и возвращает массив

В отличие от функции Input(), массив создается динамически внутри самой функции, поэтому создавать массив заранее и передавать его адрес в getNewData() не требуется

Warning
Обязательно освобождайте память, выделенную под массив, после использования!
Parameters
promptСтрока, которая напечатается в консоли для приглашения пользователя ввести числа
masМассив, в который нужно записать считанные числа
nElementsЧисло чисел, которые нужно считать. Размер переданного массива должен быть не меньше nElements

Пример использования:

const int nElements = 3;
double *coeffs = getNewData("Enter x, y, z:", nElements);
printDArray(coeffs, 0, nElements - 1);
free(coeffs);
coeffs = NULL;
