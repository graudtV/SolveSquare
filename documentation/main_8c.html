Detailed Description

Файл с примером работы с функцией SolveSquare()

Version
1.1 Теперь ввод реализуется через функцию getNewData

Программа показывает пример работы с функцией SolveSquare() а также включает возможность юнит-тестирования этой функции

Сначала проводится юнит-тест (если он включен параметром TEST_SOLVESQUARE), далее программа считывает из стандартного ввода 3 числа - коэффициенты a, b, c - и вычисляет корни соответсвующего квадратного уравнения
